🚀 Install

One line. The script installs the build dependencies with Homebrew, builds the app from source, and drops it in /Applications.

Terminal
$ curl -fsSL https://raw.githubusercontent.com/xajik/voice-to-md/main/install.sh | bash
  • Installs xcodegen, whisper-cpp and ffmpeg via Homebrew, then builds the app.
  • On first launch, grant Microphone and Accessibility access when macOS asks.
  • Download a Whisper model from Settings… in the menu bar — Base (~150 MB) is a great start.
  • Prefer to do it by hand? Follow the Quick Start on GitHub.

✨ Two modes

One hotkey for everywhere, one window for deep work.

⌨️ Global Dictation

Press ⌘⌥] in any app, speak, and the transcript is typed straight at your cursor. Terminal, browser, Slack — anything. A Spotlight-style pill shows what's happening.

📝 Agent Mode

Speak freely; a local LLM streams your words into a clean, structured markdown document in real time. The raw transcript stays one click away. Start it from the menu bar.

🧠 Dependencies & local LLM

Global Dictation only needs whisper.cpp — installed for you. Agent Mode talks to any OpenAI-compatible server running on your Mac (default http://127.0.0.1:8000/v1). Pick one:

brew install xcodegen whisper-cpp ffmpeg   # build dependencies (install.sh does this)

omlx · Apple Silicon, fastest

http://127.0.0.1:8000/v1

brew install omlx then omlx serve <model> — VTMD's default port, zero config.

Ollama

http://127.0.0.1:11434/v1

ollama pull qwen3.5 (or gemma3), then ollama serve.

LM Studio

http://127.0.0.1:1234/v1

Grab a model in the app, start the local server.

llama.cpp

http://127.0.0.1:8080/v1

llama-server -m model.gguf --port 8080.
